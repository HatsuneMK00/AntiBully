# 课程视频模块

## 开发特征

1. 每个课程视频的练习题只能在最后出现，并且只有一道题。因为目前数据库中没有为练习添加位置属性，并且能否将题目插入到视频中也存在疑问。
2. 章节和课程id编号从1开始
3. get和delete使用PathVariable的形式，post和put使用RequestParam的形式

# 题库模块

## 开发特征

1. Repo状态接口返回该Repo总题数以及用户正确题数
2. 可以获取到用户每个题库的正确题数，但是不能知道用户完成过哪些题目。take_exercise表暂时没有用。

## 疑问

1. 在点进一个题库之后，需要知道哪些题目是已经完成的吗。目前暂时是不知道的
